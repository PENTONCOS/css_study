# css_study
扎实 css 基础，实现一些复杂图形

## background

```css
{
    background: linear-gradient(#fff, #000);
}
```

1. 技巧一：渐变的颜色可以是透明色（`transparent`）
2. 技巧二：渐变可以是从一种颜色直接到另外一种颜色
3. 技巧三：渐变是可以叠加多层的
4. 技巧四：利用 `repeating-linear-gradient` 节省代码
5. 技巧五：预留衔接空间消除渐变产生的锯齿
6. 技巧六：利用多层渐变组合图形
